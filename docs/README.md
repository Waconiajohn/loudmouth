### Tech Stack
- Stem Separation: AudioShake SDK (on-device iOS/Android)
- Transcription: Whisper (OpenAI)
- Backend: FastAPI + Python 3.11
- Database: Supabase (schema in `/backend/`)
- Frontend: HTML/CSS/JS
- AI Coaching: Claude API (Anthropic)
- CDN/Hosting: Cloudflare Pages

### The Four-Step Process
1. **Listen** — Really hear the song
2. **Breathe** — Your Daily Music Meditation
3. **Blend** — Sing with the artist
4. **Own** — Pull the slider left. This is your voice now.
